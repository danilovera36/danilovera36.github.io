Case Studies

ecspresso ⭐ Destacado
Problema

Gestionar secrets y variables de entorno en múltiples servicios ECS era caótico: valores hardcodeados, rotación manual, riesgo de exposición.

Solución

Herramienta centralizada con API FastAPI para administrar secrets en AWS SSM Parameter Store + CLI interactiva para developers.

Resultado

Zero secrets hardcodeados, rotación automatizada, tiempo de setup de nuevos servicios: de horas a minutos.

env-vault
Problema

Archivos .env en repositorios Git, exposición de credenciales, sin encriptación ni rotación automática.

Solución

CLI que encripta .env con AES-256, integra con AWS KMS para gestión de claves, commiteable seguro.

Resultado

Zero credenciales en texto plano en repos, auditoría de acceso vía CloudTrail, compliance mejorado.
